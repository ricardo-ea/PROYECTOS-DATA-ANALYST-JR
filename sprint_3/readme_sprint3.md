# EXPLORACIÓN Y PROCESAMIENTO DE DATOS SOBRE HÁBITOS DE ESCUCHA DE MÚSICA EN 2 CIUDADES DIFERENTES
En este proyecto realice:

- Describi los datos que el proyecto me brindó y requirió a través de observaciones
- Procesé datos: limpieza de datos
- Analicé los datos procesados para sacar algunas primeras conclusiones 

El objetivo de este proyecto fue trabajar con la información proporcionada en una base de datos con datos reales de música online para explorar y procesar información sobre los hábitos de escucha de los y las usuarias en 2 ciudades: Springfield y Shelbyville.
