# PROYECTO DE ANALISTA PARA ZUBER "EMPRESA DE VIAJES" ENFOCADO EN EL USO GENERL DE SQL
En este proyecto trabaje en:

- Recupere datos de fuentes en linea 
- Trabaje con JSON
- Trabaje con SQL
- Aprendi a realizar requests
- Aprendi a user el metodo DISTINCT
- Aprendi a realizar conversiones de fechas
- Trabaje con el comando HAVING
- Trabaje con el comando SUM
- Trabaje con el comando WHERE
- Realice uniones con INNER JOIN, LEFT JOIN
- Realice analisis exploratorio
- Realice prueba de hipotesis

El objetivo fue encontrar patrones en la informacion disponible para comprender las preferencias de los pasajeros y el impacto de los factores externos en los viajes
