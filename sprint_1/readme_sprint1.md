# PROYECTO PROGRAMA DE FIDELIZACIÓN DE CLIENTES STORE 1
En este proyecto realicé:

- Limpieza de perfiles de clientes
- Estandaricé nombres y edades
- Calculé el gasto total por cliente
- Validé la consistencia de los datos y corregí errores
- Preparé los datos para generar indicadores de negocios KPIs

El objetivo de este proyecto fué:

- Diseñar campañas personalizadas y optimizar futuras acciones de marketing
- Para el éxito, tuvimos que segmentar a los clientes según variables como la edad, el historial de compras, y categorías de productos adquiridos.
