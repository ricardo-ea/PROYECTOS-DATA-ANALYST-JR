# APLICION WEB DE ANALISIS DE VEHICULOS
En este proyecto trabaje en:

- Configure mi perfil de GitHub creando un repositorio
- Configure mi perfil y cuenta de Render
- Realice un analisis exploratorio de datos
- Construi histogramas
- Realice graficos de dispersion
- Cree un entorno virtual
- Trabaje con VS Code
- Cree una app
- Trabaje sobre la terminal de mi ordenador
- Descargue Anaconda 
- Trabaje sobre la terminal de PowerShell
- Trabaje con los comandos de git para actualizar mi repositorio y lograr que mis carpetas se visualizaran correctamente

El objetivo de este proyecto fue brindarme la posibilidad de practicar las tareas habituales de la ingenieria en software.
