# PROYECTO COMO ANALISTA DE EMPRESA DE TELECOMUNICACIONES MEGALINE
En este proyecto realice diferentes actividades:

- Estudie y analice la informacion proporcionada de las 5 tablas
- Prepare los datos para que obtuvieran el tipo de archivo necesario y eliminé errores
- Analice los datos ya preparados previamente
- Realice la prueba de hipótesis 
- Redacte conclusiones generales

El objetivo de este proyecto fue descubrir que plan de los que comercializa la compañía genera más ingresos para ajustar el presupuesto de publicidad.
